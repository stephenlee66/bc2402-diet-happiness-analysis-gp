# BC2402 Diet and Happiness Analysis

Group project for BC2402 Designing and Developing Databases, exploring links between diet, fast food, and national happiness.

## Overview
This project analyses country-level dietary patterns, fast-food nutrition, and happiness data using SQL and NoSQL. The goal was to investigate whether diet-related patterns are associated with well-being and what broader factors may explain the results.

## Data Source
Data provided as part of the project.

## Methodology
- SQL and NoSQL Querying
- Aggregation and Filtering
- Multi-table Analysis
- Trend and Comparative Analysis
- Interpretation of Analytical Findings

## Highlights
- Analysed relationships between nutrient intake, fast-food patterns, and happiness.
- Compared long-term dietary trends with broader well-being indicators.
- Found that socioeconomic development appeared to explain more than diet alone.
