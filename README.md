# BC2402 Group Project: Diet, Fast Food, and Happiness Analysis

This group project explored how national dietary patterns, fast-food nutrition, and happiness may be related. Using both SQL and NoSQL, we analysed datasets on country-level nutrient intake, happiness scores, and menu nutrition from major fast-food chains to identify patterns across health, diet, and well-being.

## My contributions

I worked on the following questions:

- **Q11** – identified which Burger King categories were most Weight Watchers-friendly by cleaning duplicated menu records and comparing average Weight Watchers scores across categories.
- **Q12** – listed the top 10 most caloric Burger King menu items after cleaning duplicate rows and converting calorie values for correct numeric sorting. 
- **Q16** – analysed whether countries with higher average winter protein and fat intake tended to report higher happiness, using hemisphere-based winter definitions and country-level happiness joins. The conclusion was that winter eating patterns did not reliably predict national happiness. 
- **Q22** – contributed to the blue-sky analysis on unhealthy food spike months and healthier fast-food promotion strategy, as recorded in the task allocation sheet. 

## What I did

In this project, I used SQL and NoSQL to:

- clean duplicated fast-food menu records before analysis
- convert text-based nutrition values into numeric format for correct aggregation and sorting
- compare menu categories using average nutrition-related scoring
- identify high-calorie menu items through ranked queries
- merge seasonal food intake data with happiness data
- define winter periods differently by hemisphere to improve analysis logic
- interpret patterns carefully without assuming that correlation meant causation
